An advanced product filtering plugin for WooCommerce, allowing users to filter products by taxonomy and custom field.

== Description ==

The **Product filter based on custom field and custom taxonomy for woocommerce** plugin is a powerful tool for WooCommerce stores, offering an intuitive interface for customers to filter products by various criteria such as price, categories, attributes, tags, and more. This plugin enhances the shopping experience by allowing real-time filtering with AJAX technology, ensuring faster load times and a seamless user experience.

**Key Features:**

- AJAX-powered filtering for instant results without page reloads.
- Multiple filter types: checkboxes, radio buttons, dropdowns, sliders, and more.
- Filter by product categories, tags, price, attributes, and custom taxonomies.
- Compatible with popular themes and customizable to match your storeâ€™s design.
- Responsive and mobile-friendly.
- Easily integrates with WooCommerce product archives and shop pages.

This plugin is perfect for any WooCommerce store looking to improve user experience and increase sales by helping customers find exactly what they need quickly.

== Installation ==

1. Upload the `woo-custom-filter-by-webnotics.zip` file via the WordPress admin panel.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to `WooCustomFilter` menu in admin to configure your filter settings.
4. Add the filter to your shop or archive pages using the provided shortcode [WCPF_shop_page_let_sidebar].

== Frequently Asked Questions ==

= How do I add the filter to my shop page? =

You can add the filter using the provided shortcode `[WCPF_shop_page_let_sidebar]` 

= Does this plugin support custom post types? =

Yes, the plugin supports filtering for custom post types, allowing you to filter any post type in addition to WooCommerce products.

= Is the plugin compatible with all themes? =

The plugin is designed to work with most themes. However, some custom themes might require additional styling adjustments to ensure full compatibility.
